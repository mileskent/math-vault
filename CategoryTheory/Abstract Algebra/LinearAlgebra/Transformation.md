A function between [[Space|Spaces]], [[Set|Sets]], and [[Mathematical Object|Objects]]
# 1-1
Iff there is at most one location in the codomain for every location in the domain
# Onto
Iff here is a location in the codomain for every location in the domain
# Bijective
If there is exactly one output for every input of the transformation or the transformation is 1-1 and Onto
