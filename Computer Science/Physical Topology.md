How [[Network Node]]s are physically arranged

# Bus Topology
a physical topology where network nodes are connected to a single cable or bus
![[Pasted image 20251113113245.png|200]]
# Ring Topology
a physical topology where each node is connected to exactly one other node, resembling a ring
![[Pasted image 20251113113316.png|200]]
# Star Topology
a physical topology where each node is connected to a central node, resembling a star or a hub with spokes
![[Pasted image 20251113113510.png|200]]
# Mesh Topology
a physical topology where each node is connected to every other node in a network
![[Pasted image 20251113113545.png|200]]

# Hybrid Topology
a physical topology where at least two different physical topologies are combined
