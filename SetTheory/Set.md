# Operators

## Binary
### Equality
Let $A, B$ be sets
$$
\begin{gathered}
A = B\\
\iff\\
\forall x (x \in A \iff x \in B)\\
\iff\\
A \subseteq B \land B \subseteq A
\end{gathered}
$$
### Intersection
$$
A \cap B = \{x\ |\ x \in A \lor x \in B\}
$$
### Union
$$
A \cup B = \{x\ |\ x \in A \land x \in B\}
$$
### Relative Complement
$$
A - B = \{x\ |\ x \in A \land x \not \in B\}
$$
